🚀 AI/ML Internship Projects Portfolio

This repository contains a collection of four practical AI/ML projects developed as part of my internship. These projects demonstrate skills in Machine Learning, Computer Vision, and Natural Language Processing with real-world applications.

📌 Projects Overview
1️⃣ Face Mask Detection
Detects whether a person is wearing a mask or not using live camera input
Built using Computer Vision techniques
Tech Used: OpenCV, CNN, TensorFlow/Keras
2️⃣ Fake News Detection
Classifies news articles as Real or Fake
Uses NLP techniques for text processing and classification
Tech Used: TF-IDF, Logistic Regression / Naive Bayes, Scikit-learn
3️⃣ Student Performance Predictor
Predicts student performance based on factors like study hours, attendance, and previous scores
Demonstrates regression-based machine learning
Tech Used: Linear Regression, Pandas, Scikit-learn
4️⃣ AI Resume Ranker
Ranks resumes based on job description using keyword matching
Simulates real-world HR screening systems
Tech Used: NLP, TF-IDF, Cosine Similarity, Python
